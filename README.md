# Sistema Aqui Hostel

## Projeto de verão do Aqui Educa

Este projeto refere-se a um sistema interno de Hostel (Hotel).



### FUNCIONALIDADES e PROPRIEDADES:

- Acomodações:
    - Nome do quarto
    - Tipo
    - Quantidade de cama


- Funcionários
    - Nome
    - CPF
    - E-mail
    - Telefone
    
    
- Hóspede
    - Nome
    - CPF
    - E-mail
    - Telefone
    
    
- Valor
    - Acomodação
    - Valor diária
    
    
- Reserva
    - Nome hóspede
    - CPF
    - E-mail
    - Telefone
    - Acomodação
    - Data entrada
    - Data saída
    

- Pacotes
    - Nome do pacote
    - Período / data
    - Valor
    
    
- Eventos
    - Título do evento
    - Data
    - Horário
    - Descrição
    
    
- Depoimentos
    - Nome
    - Depoimento
    
    
- Atendimento
    - Nome
    - E-mail
    - Telefone
    - Mensagem
